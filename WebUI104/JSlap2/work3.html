Philosophy

Software Engineering is the applied science which aims for producing the quality software and software system with the international standard. The quality software increases the competitiveness of the international industry. Software engineers require the several skills set such as mathematic and algorithm, software design and programming, software project management, and international communication in order to respond to the international requirement. The developed software should conform to the international standard such as CMMI, and ISO 15504

The software engineering program is designed for students to study theoretically in lecture room and laboratory, and also develop the skill set using the co-operative study with the software industry. The students can apply their software engineering knowledge in the software entrepreneur to prepare themselves to the software industry. The graduated students are aim to be the competent software engineer or international software entrepreneur.
